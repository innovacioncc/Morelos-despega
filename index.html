La "Caja de Herramientas" AFDS

La propuesta educativa de Airbus Foundation Discovery Space se basa en un enfoque de aprendizaje basado en proyectos (PBL). Nuestra "Caja de Herramientas" metodológica equipa a los facilitadores con secuencias didácticas validadas internacionalmente, adaptadas para funcionar en contextos de conectividad y recursos iniciales limitados en los EMSAD de Morelos.

  • Ideación Local: Identificación de problemas de la comunidad.
  • Diseño Aeroespacial: Aplicación de principios de diseño iterativo inspirados en ingeniería de vuelo.
  • Prototipado FRUGAL: Construcción de soluciones utilizando electrónica accesible (Arduino, sensores genéricos) y materiales reciclados.

Snippets Destacados: Lectura de Humedad (Arduino)

Código base utilizado en el proyecto "Agro-Tech Morelos" para la captura de datos de humedad analógica.

int sensorPin = A0; // Pin del sensor de humedad
int sensorValue = 0;

void setup() {
  Serial.begin(9600);
}

void loop() {
  sensorValue = analogRead(sensorPin);
  Serial.println(sensorValue);
  delay(1000); // Leer cada segundo
}
